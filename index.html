StreamPing

Kirim Notif
Live Streaming
ke Discord

Tanpa coding. Langsung kirim. Super simpel.

1
Buat Webhook di Discord

Buka Server → Edit Channel → Integrations → Webhooks → New Webhook. Copy URL-nya.

2
Isi form di bawah

Paste URL webhook, isi info stream kamu, sesuaikan tampilannya.

3
Klik Kirim Notifikasi

Notif langsung muncul di channel Discord kamu. Selesai! 🎉
